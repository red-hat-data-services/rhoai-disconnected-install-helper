# XKS Charts - Disconnected Install Image Lists

This directory contains generated skopeo sync YAML files for mirroring RHAI on XKS Helm chart images in disconnected environments.

## Files

Each file is named `rhai-on-xks-chart-<VERSION>.yaml` and contains the complete list of container images (with SHA256 digests) needed for that version.

## How to Use

```bash
# Authenticate to source and destination registries
skopeo login registry.redhat.io
skopeo login <destination-registry>

# Mirror all images for a version
skopeo sync --src yaml --dest docker charts/rhai-on-xks-chart-<version>.yaml <destination-registry>
```

## How Files Are Generated

These files are auto generated by the [GitHub Actions workflow](../.github/workflows/xks-charts-disconnected-helper.yml).

For details on how images are discovered and extracted, see the [Extraction Process](../README.md#extraction-process) section in the project README.
